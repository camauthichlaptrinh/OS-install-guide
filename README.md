# OS-install-guide
hướng dẫn cài đặt hệ điều hành cho máy tính

cần usb >16gb, kết nối mạng internet...

Bước 1: Tải ventoy, win10-LTSC.iso, ( linuxmint.iso )..., 
	các phần mềm cần thiết cho window...

Bước 2: tạo usb boot, copy files iso, thư mục win-software và các dữ liệu cần
	thiết vào usb... (nếu có hiện cảnh báo chọn 'yes')

Bước 3: tắt secure boot trong bios, save changes và vào menu boot... 
	(phím vào bios và boot tuỳ dòng máy) 

Bước 4: cài đặt window... (nên xoá hết ổ cứng cũ)

Bước 5: vào win, kết nối mạng internet, update win đầy đủ, cài driver...

Bước 6: thiết lập window-defender, cài các phần mềm cần thiết...

bước 7: active win và office...

Hoàn thành

Hướng dẫn active win và office... cần có kết nối mạng internet...

Nhấp chuột phải vào nút Start (hoặc bấm Windows + X), chọn PowerShel (Admin)...
